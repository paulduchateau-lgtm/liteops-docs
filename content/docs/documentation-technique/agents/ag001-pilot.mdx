## 4. Flux utilisateur

1. **Upload** — L'utilisateur charge un fichier CSV/Excel ou connecte une base SQL
2. **Ingestion** — `OP-001 Data Layer` normalise les données en DataPacket
3. **Question** — L'utilisateur pose une question en langage naturel
4. **Traduction** — `OP-006 Text to SQL` convertit la question en requête SQL
5. **Exécution** — La requête est exécutée sur les données
6. **Analyse** — `OP-002 LLM` interprète les résultats et génère un rapport
7. **Visualisation** — `OP-007 Graph Gen` produit les graphiques correspondants
