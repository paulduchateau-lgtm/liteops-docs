# Programme Partenaires

<Callout title="Le partenaire pense, Lite Ops construit." type="info">
Zéro commission. Zéro equity. Un écosystème où le partenaire apporte l'expertise métier et conserve 100% de ses revenus. Lite Ops fournit les licences et développe les composants custom.
</Callout>

## En bref

| | Lite Ops | Partenaire |
|---|---|---|
| **Rôle** | Éditeur + dev custom | Audit / cadrage / expertise métier |
| **Revenus** | Licences SaaS + dev custom | Prestations conseil + expertise |
| **Commission versée** | 0 EUR | 0 EUR |
| **Part conservée** | 100% | 100% |
